AI Document Verification API

A powerful API for verifying identity documents using AI-powered OCR and rule-based validation. Submit documents via URL, and receive structured verification results with extracted data, confidence scores, and issue reports.

URL Push Model

Push document URLs to our API. No file uploads needed. We fetch, analyze, and return results.

AI-Powered OCR

OpenAI Vision extracts fields from documents. Custom validation rules verify extracted data.

Bulk Processing

Submit up to 50 documents in a single API call. Track batch progress in real-time.

Base URL

http://localhost:3000

Authentication

The API uses two authentication methods:

API Key (Primary)

Used for all verification endpoints. Pass your API key as a Bearer token.

Authorization: Bearer vk_your_api_key_here

JWT Token

Used for account management (profile, key generation). Obtained via login.

Authorization: Bearer eyJhbGciOi...
Integration Flow: Register → Login (get JWT) → Generate API Key (using JWT) → Use API Key for all verification calls.

  }'
2

Login & Get JWT Token

curl -X POST http://localhost:3000/v1/auth/login \
  -H "Content-Type: application/json" \

    "email": "john@example.com",
    "password": "securepassword123"

# Response: { "token": "eyJ...", "user": { ... } }

Generate an API Key

curl -X POST http://localhost:3000/v1/auth/api/generate \

  -H "Authorization: Bearer YOUR_JWT_TOKEN" \

Error Handling

All errors return a consistent JSON structure:

{
  "error": "Error type",
  "message": "Human-readable description of what went wrong"
}
StatusMeaningCommon Causes
400Bad RequestMissing required fields, invalid document_type, unsupported format
401UnauthorizedMissing or invalid API key / JWT token
403ForbiddenAccessing another user's resource, insufficient role
404Not FoundVerification request or resource not found
409ConflictDuplicate document type code
429Too Many RequestsRate limit exceeded (hourly or burst)
500Server ErrorInternal error, contact support

Rate Limits

Each API key has configurable rate limits (set during key generation):

Hourly Limit

Default: 1,000 requests/hour

Burst Limit

Default: 50 requests/minute

When rate limited, you'll receive a 429 response. Wait for the limit window to reset before retrying.

Verification Endpoints

POST /v1/verify Submit single document for verification

Requires API Key auth. Async - returns immediately with a reference ID. Poll status/result endpoints for outcome.

Request Body

{
  "document_type": "string (required) - e.g. 'aadhaar', 'pan', 'passport'",
  "file_url": "string (required) - Public URL to the document image/PDF",
  "reference_id": "string (optional) - Your own tracking ID",
  "metadata": {
    "key": "value (optional) - Cross-verification data e.g. applicant name"
  }
}

Response 202 Accepted

{
  "status": "accepted",
  "system_reference_id": "DOCLXYZ12345ABC",
  "processing_status": "processing"
}

Available Document Types

aadhaar pan passport driving_license voter_id bank_statement utility_bill marksheet_10 marksheet_12 graduation_cert + your custom types

Verifying Signatures (Node.js)

const crypto = require('crypto');

function verifyWebhookSignature(payload, signature, secret) {
  const expected = 'sha256=' + crypto
    .createHmac('sha256', secret)
    .update(JSON.stringify(payload))
    .digest('hex');
  return crypto.timingSafeEqual(
    Buffer.from(signature),
    Buffer.from(expected)
  );
}

Integration Code Samples

Node.js Integration

const API_KEY = 'vk_your_api_key';
const BASE_URL = 'http://localhost:3000';

// Single document verification
async function verifyDocument(docType, fileUrl, referenceId, metadata) {
  const res = await fetch(`${BASE_URL}/v1/verify`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${API_KEY}`
    },
    body: JSON.stringify({
      document_type: docType,
      file_url: fileUrl,
      reference_id: referenceId,
      metadata
    })
  });
  return res.json();
}

// Bulk verification
async function bulkVerify(documents, callbackUrl) {
  const res = await fetch(`${BASE_URL}/v1/verify/bulk`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${API_KEY}`
    },
    body: JSON.stringify({ documents, callback_url: callbackUrl })
  });
  return res.json();
}

// Poll for result
async function waitForResult(systemRefId, maxWaitMs = 60000) {
  const start = Date.now();
  while (Date.now() - start < maxWaitMs) {
    const res = await fetch(`${BASE_URL}/v1/result/${systemRefId}`, {
      headers: { 'Authorization': `Bearer ${API_KEY}` }
    });
    const data = await res.json();
    if (!['accepted', 'processing'].includes(data.status)) return data;
    await new Promise(r => setTimeout(r, 2000));
  }
  throw new Error('Timeout waiting for result');
}

// Example usage
(async () => {
  // Single
  const single = await verifyDocument('aadhaar',
    'https://s3.example.com/aadhaar.jpg', 'TXN_001',
    { applicant_name: 'Rahul Sharma' });
  console.log('Submitted:', single.system_reference_id);

  const result = await waitForResult(single.system_reference_id);
  console.log('Result:', result.status, result.extracted_data);

  // Bulk
  const bulk = await bulkVerify([
    { document_type: 'aadhaar', file_url: 'https://s3.example.com/doc1.jpg', reference_id: 'APP_001' },
    { document_type: 'pan', file_url: 'https://s3.example.com/doc2.jpg', reference_id: 'APP_002' },
    { document_type: 'passport', file_url: 'https://s3.example.com/doc3.pdf', reference_id: 'APP_003' }
  ], 'https://your-server.com/bulk-callback');
  console.log('Bulk submitted:', bulk.bulk_id);
})();

Python Integration

import requests
import time

API_KEY = "vk_your_api_key"
BASE_URL = "http://localhost:3000"
HEADERS = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {API_KEY}"
}

# Single document verification
def verify_document(doc_type, file_url, reference_id=None, metadata=None):
    payload = {"document_type": doc_type, "file_url": file_url}
    if reference_id: payload["reference_id"] = reference_id
    if metadata: payload["metadata"] = metadata
    resp = requests.post(f"{BASE_URL}/v1/verify", json=payload, headers=HEADERS)
    return resp.json()

# Bulk verification
def bulk_verify(documents, callback_url=None):
    payload = {"documents": documents}
    if callback_url: payload["callback_url"] = callback_url
    resp = requests.post(f"{BASE_URL}/v1/verify/bulk", json=payload, headers=HEADERS)
    return resp.json()

# Poll for result
def wait_for_result(system_ref_id, max_wait=60):
    start = time.time()
    while time.time() - start < max_wait:
        resp = requests.get(f"{BASE_URL}/v1/result/{system_ref_id}", headers=HEADERS)
        data = resp.json()
        if data.get("status") not in ("accepted", "processing"):
            return data
        time.sleep(2)
    raise TimeoutError("Timeout waiting for result")

# Example
result = verify_document("aadhaar", "https://s3.example.com/aadhaar.jpg",
    reference_id="TXN_001", metadata={"applicant_name": "Rahul Sharma"})
print(f"Submitted: {result['system_reference_id']}")

final = wait_for_result(result["system_reference_id"])
print(f"Status: {final['status']}, Data: {final.get('extracted_data')}")

# Bulk
bulk = bulk_verify([
    {"document_type": "aadhaar", "file_url": "https://s3.example.com/doc1.jpg"},
    {"document_type": "pan", "file_url": "https://s3.example.com/doc2.jpg"},
])
print(f"Bulk submitted: {bulk['bulk_id']}")
